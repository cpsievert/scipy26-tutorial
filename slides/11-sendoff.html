The recipe

  1. Focus LLMs on language → code
  2. Give them tools and context
  3. Keep humans in the loop

That’s it. Context engineering + tools + verification.

No RAG. No multi-agent. No fine-tuning.

Repeat the thesis from the foundations section. The whole tutorial was an exercise in showing how far you can get with these three simple principles. querychat is the proof: a production-ready analytics platform built entirely on context engineering (auto-generated schema) + tools (filter, query, visualize) + human-in-the-loop (verifiable SQL, reactive dashboards).
